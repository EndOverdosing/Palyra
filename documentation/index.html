1. Getting Started

Welcome to Palyra! This documentation will help you understand and make the most of our secure communication platform.

1.1 Creating an Account

  1. Visit the Palyra website and click "Sign Up".
  2. Choose a unique username (3-20 characters, letters, numbers, underscores, and hyphens only).
  3. Create a strong password (minimum 8 characters with uppercase, lowercase, and numbers).
  4. Click "Sign Up" to create your account.

Note: Your username is permanent and visible to other users. Choose carefully!

1.2 Logging In

Enter your username and password on the login page. For security, accounts are locked for 15 minutes after 5 failed login attempts.

2. User Interface Overview

2.1 Sidebar

  • Profile Section: Shows your avatar and username. Click settings icon to customize.
  • Notifications: Bell icon displays unread notifications count.
  • Add Friend: Plus icon to send friend requests.
  • Conversations List: Shows all your active chats and servers.
  • Navigation Tabs: Switch between "Personal" (direct messages) and "Global" (servers).

2.2 Chat Area

  • Header: Displays friend/server name, status, and action buttons (call, options).
  • Messages: Scrollable message history with timestamps.
  • Input Bar: Type messages, attach files, and send messages.

3. Friend Management

3.1 Adding Friends

  1. Click the "+" button in the sidebar.
  2. Enter the exact username of the person you want to add.
  3. Click "Send Request".
  4. Wait for them to accept your request.

3.2 Managing Friend Requests

  • Click "Pending Requests" in your conversations list to see incoming requests.
  • Accept or decline requests with the checkmark or X buttons.
  • Declined requests are permanently removed.

3.3 Blocking Users

  1. Open a chat with the user.
  2. Click the three-dot menu in the chat header.
  3. Select "Block User".
  4. Confirm the action.

Blocked users cannot send you messages or see your online status. Manage blocked users in Settings ‚Üí Privacy.

4. Messaging Features

4.1 Sending Messages

  • Type in the input box at the bottom of the chat.
  • Press Enter to send (Shift+Enter for new line).
  • Messages support plain text and URLs (auto-linked).

4.2 File Sharing

  1. Click the paperclip icon in the input bar.
  2. Select one or more files (images, videos, documents, etc.).
  3. Preview appears above the input bar.
  4. Remove files by clicking the X on each preview.
  5. Add a message (optional) and send.

File Limits: Maximum 25MB per file. Images over 5MB are automatically compressed.

4.4 Link Previews

When you send a URL, Palyra automatically generates a preview with title, description, and thumbnail (when available). Powered by Microlink API.

4.5 Typing Indicators

See when your friend is typing a message. The indicator appears at the bottom of the chat and disappears after 3 seconds of inactivity.

5. Video & Audio Calls

5.1 Starting a Call

  1. Open a chat with a friend.
  2. Click the video camera icon in the chat header.
  3. Grant camera and microphone permissions when prompted.
  4. Wait for your friend to accept the call.

Important: Video calls require HTTPS. Use localhost for testing or deploy with SSL.

5.2 Call Controls

  • Mute/Unmute: Toggle microphone on/off.
  • Video On/Off: Enable or disable your camera.
  • Screen Share: Share your screen (desktop only).
  • End Call: Hang up and return to chat.
  • Camera Flip: Switch between front/back camera (mobile only).

5.3 Call Quality

Calls use peer-to-peer WebRTC technology with automatic quality adjustment based on network conditions. For best results:

  • Use a stable Wi-Fi or wired connection.
  • Close unnecessary browser tabs and applications.
  • Ensure good lighting for video calls.
  • Use headphones to prevent echo.

6. Servers (Group Chats)

6.1 Creating a Server

  1. Navigate to the "Global" tab.
  2. Click the "+" button.
  3. Enter a server name.
  4. Choose public or private (password-protected).
  5. Click "Create Server".

6.2 Joining Servers

  • Public Servers: Click any server in the list to join instantly.
  • Private Servers: Enter the password when prompted.

6.3 Server Management (Owner Only)

As a server owner, you can:

  • View Members: See all server participants.
  • Kick Members: Remove disruptive users.
  • Delete Server: Permanently remove the server and all its messages.

Access these options by clicking the info icon in the server chat header.

6.4 Leaving Servers

  1. Open the server chat.
  2. Click the three-dot menu.
  3. Select "Leave Server".
  4. Confirm the action.

7. Settings & Customization

7.1 Profile Settings

  • Change Avatar: Click your avatar in settings and upload an image (max 25MB, auto-compressed).
  • Change Username: Click "Change Username" and enter a new unique username.

7.2 Appearance

  • Dark Mode: Toggle between light and dark themes.
  • Font Size: Choose Small, Medium, or Large text size.

7.3 Notifications

  • Enable Notifications: Receive browser notifications for new messages.
  • Message Sounds: Play sound effects when receiving messages.

7.4 Privacy

  • Show Online Status: Let friends see when you're online.
  • Read Receipts: Let friends know when you've read their messages.
  • Blocked Users: View and manage your blocked users list.

8. Security & Privacy

8.1 End-to-End Encryption

All video and audio calls are end-to-end encrypted using WebRTC technology. Palyra servers never see or store call content‚Äîonly connection metadata (duration, participants).

8.2 Security Features

  • Password Hashing: Passwords are never stored in plain text.
  • Rate Limiting: Prevents spam and abuse (100 messages/minute, 10 friend requests/5 minutes).
  • Bot Detection: Honeypot fields and interaction tracking prevent automated abuse.
  • Fingerprinting: Device fingerprints help detect suspicious activity.
  • Account Lockout: 5 failed login attempts = 15-minute lockout.

8.3 Data Storage

  • Text messages are stored on secure Supabase servers.
  • Files are stored in Supabase Storage with public URLs.
  • Video/audio call content is NEVER stored‚Äîonly peer-to-peer.
  • Account data is deleted within 30 days of account deletion.

9. Troubleshooting

9.1 Can't Connect to Video Call

  • Ensure you're using HTTPS (not HTTP).
  • Grant camera/microphone permissions in browser settings.
  • Check if another app is using your camera.
  • Try a different browser (Chrome, Firefox, or Edge recommended).
  • Disable VPN or firewall temporarily.

9.2 Messages Not Sending

  • Check your internet connection.
  • Refresh the page (Ctrl+R or Cmd+R).
  • Clear browser cache and cookies.
  • Verify you haven't hit rate limits (100 messages/minute).

9.3 Friend Request Not Arriving

  • Verify you entered the exact username (case-sensitive).
  • Check if the user has blocked you.
  • Wait a few minutes and check notifications again.
  • Ask them to check their "Pending Requests" section.

9.4 File Upload Failing

  • Ensure file is under 25MB.
  • Check supported file types (images, videos, documents, audio).
  • Try compressing large files before uploading.
  • Check your internet connection stability.

11. API & Development

11.1 Technology Stack

  • Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3
  • Backend: Supabase (PostgreSQL, Realtime, Storage, Auth)
  • WebRTC: PeerJS library for video/audio calls
  • Icons: Font Awesome 6

11.2 Self-Hosting

To self-host Palyra, you'll need:

  1. A Supabase project (free tier available).
  2. Node.js for development server.
  3. SSL certificate for HTTPS (required for video calls).

See the GitHub README for detailed setup instructions.
